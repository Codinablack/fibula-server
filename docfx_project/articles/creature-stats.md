# Creature stats

All creatures have stats, some exposed and some hidden, which quantify the abilities or 

Creatures in the game have the following basic stats available:

| Name | Type | Hidden |
| -- | -- | -- |
| HitPoints | int | No |
| ManaPoints | int | No |
| SoulPoints | int | No |
| Capacity | int | No |
| BaseSpeed | int | No |
| AttackPoints | int | Yes |
| DefensePoints | int | Yes |
